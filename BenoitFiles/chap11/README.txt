medprof.m
Script that provides temperature and salinity profiles in the Mediterranean Sea, horizontally
averaged in box defined by the user. The profiles can be further used to calculate density with the help 
of ies80.m

avrprof.m
Function that reads profiles and averages them in the box provided as input

medarncread.m
Function which reads the netCDF file containing the MEDAR Mediterranean climatology

ies80.m
Function which allows to calculate ocean water density from temperature, salinity and pressure
